# İks - Hızlı Başlangıç Kılavuzu

## 🚀 5 Dakikada İks'i Başlat!

### Adım 1: Başlat (30 saniye)
```bash
# Windows
start.bat

# Diğer
npm start
```

### Adım 2: Kayıt Ol (1 dakika)
1. Uygulama açılınca "Kayıt Ol" sekmesine tıkla
2. Bilgileri doldur:
   - **Kullanıcı adı**: benzersiz bir isim
   - **Email**: geçerli bir email
   - **Görünen ad**: gerçek adın
   - **Şifre**: güvenli bir şifre
3. "Kayıt Ol" butonuna tıkla

### Adım 3: Giriş Yap (30 saniye)
1. "Giriş Yap" sekmesine geç
2. Kullanıcı adını ve şifreni gir
3. "Giriş Yap" butonuna tıkla
4. **Artık asla tekrar giriş yapman gerekmeyecek!** ✨

### Adım 4: İlk İksini Paylaş (1 dakika)
1. "Ne düşünüyorsun?" kutusuna tıkla
2. Bir şeyler yaz, örneğin:
   ```
   Merhaba İks! Bu benim ilk paylaşımım 🎉 #yenibaşlangıç #iks
   ```
3. İsteğe bağlı: 🖼️ ikonuna tıklayarak fotoğraf ekle
4. "İksle" butonuna tıkla
5. **Tebrikler! İlk iksini paylaştın!** 🎊

### Adım 5: Keşfet (2 dakika)
1. Sol menüden "Keşfet"e tıkla
2. Arama kutusunu kullan
3. Kullanıcıları bul ve takip et
4. Trendleri incele

## 🎯 Hızlı İpuçları

### Hashtag Kullan
```
#teknoloji #yapayZeka #kodlama
```
# ile başlayan kelimeler otomatik mavi renkte görünür!

### Medya Yükle
- 🖼️ ikonuna tıkla
- Fotoğraf veya video seç
- Önizleme görünür
- Paylaş!

### Tema Değiştir
1. Sol menüden "Ayarlar"a git
2. "Açık" veya "Koyu" tema seç
3. Anında değişir!

### Mesajlaş
1. Sol menüden "Mesajlar"a git
2. Bir kullanıcı seç
3. Mesaj yaz ve gönder
